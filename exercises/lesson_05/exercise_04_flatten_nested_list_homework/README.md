# Exercise 04 — Flatten a Nested List (Homework)

Write `flatten(lst)` that takes a list which may contain other lists
nested to any depth — for example `[1, [2, [3, [4]], 5]]` — and returns a
single flat list, e.g. `[1, 2, 3, 4, 5]`.

Use `isinstance(item, list)` to check whether an item is itself a list,
and recurse into it if so.
